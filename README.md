# 🪙 GOLDEN CAISSE - Golden Palace Hôtel

**GOLDEN CAISSE** est une application web moderne de gestion administrative, financière et de suivi de caisse conçue spécifiquement pour le **Golden Palace Hôtel**. Ce système offre une interface épurée et performante pour la gestion des transactions, le transfert de fonds et la régularisation comptable.

---

## 🚀 Fonctionnalités Clés

* **🔒 Double Environnement d'Accès :**
  * **Caissière :** Ouverture/fermeture de caisse avec billetage physique, saisie des opérations d'entrée/sortie, gestion des pièces justificatives manquantes, transferts de fonds inter-caisses.
  * **Responsable Administratif et Financier (RAF) :** Tableau de bord analytique (flux financiers entrées/sorties via graphiques dynamiques), validation des régularisations, historique global des transactions et configuration de l'établissement.
* **📈 Bilans en Temps Réel :** Suivi automatisé des soldes de la **Caisse Principale** et de la **Caisse d'Exploitation**.
* **🧾 Gestion des Justificatifs :** Système d'alerte visuel pour les pièces comptables en attente de visa ou de justificatifs.
* **🖨️ Édition de Rapports :** Génération et impression automatique des rapports de clôture journaliers et des états de caisse.
* **🌓 Mode Sombre Natif :** Interface moderne, sombre et premium avec des accents dorés (gold).

---

## 🛠️ Stack Technique

* **Frontend :** HTML5 sémantique, CSS3 moderne (variables personnalisées, design adaptatif), JavaScript (ES6+).
* **Graphiques :** Chart.js pour la visualisation des flux.
* **Icônes :** Font Awesome v6.
* **Base de données :** Système de base de données locale (stockage structuré et persistant).

---

## 📋 Installation et Démarrage Local

1. Clonez ce dépôt sur votre machine locale :
   ```bash
   git clone https://github.com/JeanDonald501/GloryPressManagement.git
   ```
2. Ouvrez le dossier du projet :
   ```bash
   cd GloryPressManagement
   ```
3. Ouvrez le fichier `index.html` dans votre navigateur ou lancez un serveur local (ex: *Live Server* dans VS Code).
